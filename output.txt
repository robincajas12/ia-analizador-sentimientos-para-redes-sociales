He corregido el archivo `analizador-de-sentimientos-con-ia/src/app/actions.ts`.

El problema era que el frontend estaba intentando conectar a una dirección `http://api:5001/predict` codificada, en lugar de usar la variable de entorno `NEXT_PUBLIC_API_URL` que se configuró en `docker-compose.yaml`.

He modificado la línea de `fetch` para que use correctamente la variable de entorno:
De: `const response = await fetch('http://api:5001/predict', {`
A: `const response = await fetch(`${process.env.NEXT_PUBLIC_API_URL}/predict`, {`

Por favor, intenta levantar las aplicaciones de nuevo con:

```bash
docker compose up --build
```
